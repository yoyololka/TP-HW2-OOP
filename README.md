# Система управления рецептами

Учебный проект по курсу "Технологии программирования", НИУ ВШЭ, 2025-26.

Реализация системы управления рецептами: создание блюд, добавление ингредиентов, масштабирование порций и генерация списка покупок.

## Установка

```bash
git clone https://github.com/yoyololka/TP-HW2-OOP.git
cd TP-HW2-OOP
pip install -r requirements.txt
```

## Использование

Классы находятся в файле `recipes.py`. Для запуска тестов активируйте виртуальное окружение и запустите pytest:


# Windows
```bash
.\venv\Scripts\activate
pytest
```
